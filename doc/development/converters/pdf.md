# PDF Converter

## Features
- PDF to Image conversion
- Image to PDF conversion
- PDF merge/split
- PDF compression
- Basic PDF editing
- OCR (text recognition)

## Implementation
- Using pdf-lib for manipulation
- jsPDF for creation
- Tesseract.js for OCR
- Browser Canvas for preview
